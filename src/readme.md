A simple script that interacts with the [NMEA Simulator](https://github.com/panaaj/nmeasimulator) via websocket. It gets the NMEA sentences and by a simple TTS make a corresponding voice message 

* NMEA 2000 --> BUS
* NMEA 0183 --> SERIAL
